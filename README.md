# Feliz cumpleaños mamá — versión 3 corregida

Esta versión corrige el enfoque de la versión 2. Ahora busca parecerse más al video de referencia:

- Fondo negro.
- Inicio con explosión/rayos rojos desde el centro.
- Corazón rojo/neón formado por líneas brillantes, rastros y contornos.
- Movimiento con estelas luminosas, no solo partículas sueltas.
- Se puede girar con el dedo o mouse.
- Al acumular 3 vueltas aparece:
  - “Feliz cumpleaños mamá”
  - “Te amo mucho!!!”
- El fondo se desenfoca.
- Detrás de las letras aparecen confeti y fuegos artificiales.

## Cómo probar

1. Abre esta carpeta en VS Code.
2. Usa Live Server.
3. Abre `index.html`.
4. Arrastra el corazón con el mouse o con el dedo en celular.

## Cómo actualizar en GitHub Pages

1. Entra a tu repositorio.
2. Reemplaza los archivos anteriores por los de esta carpeta.
3. Haz commit.
4. Espera unos minutos.

Si sigue saliendo la versión anterior en el celular:
- Abre el enlace en modo incógnito, o
- Borra los datos del sitio, o
- Añade `?v=3` al final del enlace.
