# Feliz cumpleaños mamá — versión 2

Versión más parecida al video de referencia: corazón rojo/neón creado con partículas, líneas brillantes, rayos desde el centro y un efecto más oscuro/cinemático.

## Qué incluye

- Corazón animado con partículas y líneas rojas.
- Efecto de rayos desde el centro, parecido al inicio del video.
- Rotación táctil con el dedo en celular.
- Al girarlo 3 vueltas aparece:
  - “Feliz cumpleaños mamá”
  - “Te amo mucho!!!”
- Fondo con blur para resaltar el texto.
- Confeti y fuegos artificiales detrás de las letras.
- Funciona sin librerías externas: no depende de Three.js ni CDN.

## Cómo probarlo

1. Abre la carpeta en VS Code.
2. Usa Live Server.
3. Abre `index.html`.
4. En PC arrastra con el mouse; en celular arrastra con el dedo.

## Cómo actualizarlo en GitHub Pages

1. Entra a tu repositorio.
2. Borra o reemplaza los archivos anteriores.
3. Sube los archivos de esta carpeta.
4. Haz commit.
5. Espera unos minutos y abre tu enlace de GitHub Pages.

Nota: si tu celular ya había abierto la versión anterior, puede tardar un poco en refrescar por el caché. Si pasa, abre el enlace en modo incógnito o borra datos del sitio.
