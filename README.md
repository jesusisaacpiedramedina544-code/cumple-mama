# Feliz cumpleaños mamá — versión 5

Esta versión corrige que el corazón del centro no se veía.

## Qué se corrigió

- Se agregó un corazón central dibujado claramente.
- Tiene relleno, borde luminoso y brillo interior.
- Sigue siendo ligera para que no se trabe tanto.
- Mantiene las líneas y el look rojo/neón alrededor.

## Cómo probar

1. Abre la carpeta en VS Code.
2. Usa Live Server.
3. Abre `index.html`.
4. Si ves caché, usa el enlace con `?v=5`.

Ejemplo:
https://tuusuario.github.io/regalo-mama/?v=5
